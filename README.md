## Quick Reference: Platform Preferences

### **EDUCATIVE (PREFERRED)** ✓
- **Why:** Interactive hands-on learning, no videos (learn at your pace)
- **Best for:** Technical depth, coding practice, real-world projects
- **Pricing:** ~$400/year or ~$30/month subscription
- **URL:** educative.io

### **DeepLearning.AI**
- **Why:** Official Deep Learning courses from creators of the specialization
- **Best for:** Neural networks, NLP, generative AI
- **Pricing:** Free for basic courses, paid specializations ~$50-100/month
- **URL:** deeplearning.ai

### **Coursera**
- **Why:** University-level courses, specializations with certificates
- **Best for:** Comprehensive learning paths, rigorous theory
- **Pricing:** ~$40-50/month or audit free (limited access)
- **URL:** coursera.org

### **Udemy**
- **Why:** Practical, project-based, lifetime access
- **Best for:** Hands-on implementation, specific tools
- **Pricing:** $12-15 per course (frequent sales), one-time purchase
- **URL:** udemy.com

## Study Strategy Recommendations

1. **Start with Educative** for your main course topics (preferred for interactive learning)
2. **Supplement with DeepLearning.AI** for neural network-heavy topics (DRL, Unsupervised DL, CV, NLP)
3. **Use Coursera specializations** for comprehensive multi-week learning paths
4. **Pick Udemy projects** for hands-on reinforcement with specific tools/frameworks

### Sample Weekly Schedule:
- **Monday-Wednesday:** Attend college lectures + Educative course content
- **Thursday:** DeepLearning.AI practice/projects
- **Friday-Saturday:** Udemy project implementation
- **Sunday:** Review + Coursera theory reading
